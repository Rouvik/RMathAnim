Class: RAPoly

RAPoly(points, translateopt, optionsopt)

Animated polygon rendering class

Constructor

new RAPoly(points, translateopt, optionsopt)

Creates an RAPoly object to render animated polygons to screen

Note: this.RPoly is used to compose the static RPoly components to the RAPoly animation class, all properties to RPoly must be accessed from "this.RPoly"

Parameters:
Name Type Attributes Default Description
points Array An array of x, y coordinates placed side by side for the polygon, eg = [x0, y0, x1, y1, x2, y2, ..., xn, yn]
translate vec2 <optional>
 new vec2(0, 0) The position vector to translate to before drawing the polygon
options Object <optional>
 {} Options to control the looks of the polygon {lineColor: string, backColor: vec3, maxAlpha: number(0 - 1)}
