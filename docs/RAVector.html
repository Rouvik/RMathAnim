Class: RAVector

RAVector(start, end, options)

Animated vector, extends the RAnimation class and uses combination this.RVector to implement RVector in it

Constructor

new RAVector(start, end, options)

Creates an animated RVector

Note: this.RVector is used to compose RVector class, all vector properties must be accessed from "this.RVector"

Parameters:
Name Type Description
start vec3 The starting position vector to the vector
end vec3 The ending position vector to the vector
options Object Options {color: vec3, arrowSize: number} used to descrive the vector
